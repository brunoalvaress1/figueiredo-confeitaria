# 🍰 Figueiredo Confeitaria

Sistema web desenvolvido para a **Figueiredo Confeitaria**, com foco em vendas online de **bolos, açaí e produtos personalizados**, oferecendo uma experiência prática e moderna para os clientes realizarem pedidos.

---

## 📌 Sobre o Projeto

O projeto foi criado para digitalizar as vendas da confeitaria, facilitando pedidos online e aumentando o alcance do negócio através de uma plataforma moderna, responsiva e integrada ao WhatsApp.

O cliente navega pelos produtos, adiciona itens ao carrinho e finaliza o pedido diretamente com a atendente.

---

## ⚙️ Funcionalidades

### 🛍️ Catálogo de Produtos

* Exibição de bolos disponíveis
* Exibição de opções de açaí
* Produtos organizados por categoria
* Visual moderno e atrativo

### 🛒 Carrinho de Compras

* Adicionar múltiplos itens
* Alterar quantidades
* Remover produtos
* Cálculo automático do total

### 📲 Pedido via WhatsApp

* Geração automática da mensagem
* Lista detalhada dos produtos
* Quantidades e valores
* Valor total do pedido
* Redirecionamento direto para o WhatsApp

### 📱 Layout Responsivo

* Compatível com celular
* Navegação simples e rápida
* Interface intuitiva

---
